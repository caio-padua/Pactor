Pasta oficial das especificacoes do sistema.

## Convencao

1. As specs sao escritas pelo Dr. Claude (orquestrador).
2. Sao numeradas em ordem (ex.: `28_dieta.md`, `29_tabela_editavel.md`, `30_orcamento.md`).
3. Regra de ferro R1: spec antes de codigo. Sem spec aqui, nao se prototipa.
4. O Dr. Code sobe as specs do Dr. Claude em `docs/dr-claude/`.
