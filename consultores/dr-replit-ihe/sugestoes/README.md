# Sugestoes do Dr. Replit IHE (producao)

Notas e pedidos do senior de producao para o laboratorio (Pactor).
Um arquivo por tema.

Sugestao de nome: `AAAA-MM-DD-tema.md` (ex.: `2026-06-13-ajuste-para-banco-real.md`).

Cada arquivo deve conter:
1. O que foi analisado.
2. O problema ou a oportunidade.
3. A sugestao concreta.
4. O impacto (o que melhora, o que pode quebrar na producao).
