# Sugestoes do Dr. Manus (missoes)

Entregas, analises e notas do engenheiro de missoes para o Pactor.
Um arquivo por tema.

Sugestao de nome: `AAAA-MM-DD-tema.md` (ex.: `2026-06-13-missao-forja-publica.md`).

Cada arquivo deve conter:
1. A missao.
2. O que foi explorado.
3. A entrega ou a recomendacao.
4. O proximo passo.
